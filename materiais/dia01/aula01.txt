Introdução ao Java

Linguagens de programação:
    - Linguagens de alto nível (ex: Pascal, C) são feitas para humanos.
    - O computador só entende linguagem de máquina.
    - É necessário um compilador para transformar código-fonte em executável.

Problema dos executáveis:
    - Um programa compilado funciona apenas no sistema para o qual foi gerado.
    - Isso dificultava rodar o mesmo programa em vários sistemas.

Linguagem C:
    - Resolveu parcialmente o problema por ser multiplataforma.
    - O mesmo código pode ser recompilado para diferentes sistemas.
    - Base de sistemas como Windows, Linux, macOS e Android.

Origem do Java:
    - Criado pela Sun Microsystems (1990).
    - Liderado por James Gosling.
    - Objetivo: permitir comunicação entre diferentes dispositivos.

Oak → Java:
    - A linguagem se chamava Oak, mas o nome já existia.
    - Foi renomeada para Java, inspirada no café forte (“Java Coffee”).

Ideia central do Java:
    - Criar programas que funcionem em qualquer plataforma.
    - Conceito: “Write once, run anywhere”.

Popularização:
    - Java ganhou força com a web e a interatividade.
    - Não confundir Java com JavaScript (são linguagens diferentes).

Java usado hoje em:
    - Android
    - Sistemas bancários
    - Cartões com chip
    - TV Digital
    - Blu-ray
    - Receita Federal
